Classe_Equivalence

Objet (graphique) représentant une classe d'équivalence.

Constructor

new Classe_Equivalence(_opt)

Source:
  • classes/CLASS-Classe_Equivalence.js, line 521
Constructeur
Parameters:
Name Type Description
_opt Object [FACULTATIF] Options de création de la classe d'équivalence. L'objet complet est de la forme : {pos:..., couleur:..., bloque:...}, mais seuls les informations strictement nécessaires peuvent être renseignées. pos est de type Position (par défaut : l'origine de SCHEMA) ; couleur est STRING ; bloque est un booléen (si la pièce est bloquée, référentiel, etc.)
